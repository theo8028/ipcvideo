# Open IPC Video

Open source ipc(interprocess communication)-based  low latency video data sharing library.

## Support Language

* C/C++
* C#/WPF
* Python

## Support OS

Tested in Windows 11 and Visual Studio 2022 environments.( Linux not yet supported )
